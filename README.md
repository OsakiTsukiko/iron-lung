# 🔧 Build and Run
### ⚙️ Prerequisites
```bash
# 🦎 Zig (0.14.0)
# https://ziglang.org/

# 👒 Sassy CSS (sass)
bun i -g sass
# or
npm i -g sass
```

### 📦 Compilation
```bash
zig build scss # 👒 compile scss
zig build run-full # 🏁 compile and run the whole appilcation
```
